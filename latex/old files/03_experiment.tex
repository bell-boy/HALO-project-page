\section{Experimental Plan}

\begin{itemize}
    \item Stress test recall of the neural memory over long horizon
    \item Validate neural memory is faster and works better than in-context learning on XLand-MiniGrid. (Contribution 1)
    \item Validate generative neural memory leads to faster in-context learning on XLand-MiniGrid. (Contribution 2)
    \item \textbf{Real-world execution via hierarchical control.} To deploy on a humanoid, we will have to develop a hierarchical architecture in which a low-level controller optimizes motion-tracking--based rewards for specified goals, while a high-level policy selects and sequences those goals. This decomposition will make RL tractable in high-dimensional action spaces and requires substantial systems engineering for robust sim2real transfer. Note the memory part remains the same. (Contribution 3)

\end{itemize}
